## About

An app to track your dog's health, training and memories.

## Features

- Secure registration & login (JWT, hashed passwords)
- Multiple dogs per account, each with their own profile
- Dashboard showing what matters now: upcoming vet visits, medications due, recent journal entries, training progress
- Full create/edit/delete for vet visits, medications, journal entries, and training exercises

## Tech Stack

- React
- TypeScript
- mongoDB
- express.js
- node.js
- Vite
- JWT
- bcrypt
- CSS
